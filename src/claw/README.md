# OpenClaw Skill Pack

[English](./README.md) | [中文](./README.zh.md)

This folder contains **skill documents for OpenClaw to learn and follow directly**, instead of code-level API wrappers.

## Included files

- `SKILL.md`: English behavior spec for OpenClaw
- `SKILL.zh.md`: Chinese behavior spec for OpenClaw

## What this solves

It tells OpenClaw how to:

1. understand the full PoreMind analysis workflow,
2. explain function purpose and parameter effects step by step,
3. produce structured outputs (summary, tables, figures, model artifacts),
4. keep the process reproducible and auditable.
