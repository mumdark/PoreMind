# OpenClaw 技能包（行为规范）

[English](./README.md) | [中文](./README.zh.md)

本目录提供的是让 OpenClaw **直接学习并遵循**的技能文档，而不是代码层 API 封装。

## 包含文件

- `SKILL.md`：英文版行为技能说明
- `SKILL.zh.md`：中文版行为技能说明

## 解决的问题

该技能文档明确了 OpenClaw 如何：

1. 理解并执行完整 PoreMind 分析流程；
2. 分步骤解释函数作用与参数影响；
3. 输出结构化分析结果（结论、表格、图像、模型产物）；
4. 保证流程可复现、可审计。
