# PoreMind Documentation

Welcome to the PoreMind docs site.

PoreMind is an API-oriented toolkit for single-molecule nanopore analysis, including denoising, event detection, feature extraction, filtering, model training, and new-sample classification.

## Documentation Structure

- **Installation**: Environment and dependency setup.
- **Quick Start**: Step-by-step analysis flow aligned with `notebooks/step_by_step_analysis.ipynb`.
- **Method Framework**: Conceptual pipeline overview.
- **Function Reference**: One file per function/method.
- **Citation & License**: How to cite and MIT usage notes.
