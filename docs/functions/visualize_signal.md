# `MultiSampleAnalysis.visualize_signal`
**Module:** `workflow.py`

Plots a selected signal window for quick visual inspection.

## Parameters
- `sample_id` (`str`): Trace id to plot.
- `start_s` (`float`): Start time in seconds.
- `end_s` (`float | None`): End time in seconds.
- `max_points` (`int`): Maximum plotted points.

## Returns
- `matplotlib Axes.`
