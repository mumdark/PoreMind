# `PlotAccessor.model_metric_bar`
**Module:** `pl.py`

Plots bar chart of weighted metrics across models.

## Parameters
- `metric` (`str`): f1, accuracy, or recall.
- `split` (`str`): train or test.
- `width` (`float`): Figure width.
- `height` (`float`): Figure height.

## Returns
- `matplotlib Axes`
